## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `VITE_MY_MAP_ID` in [.env.local](.env.local) to your MID
3. Run the app:
   `npm run dev`

Get your Map ID: Open your Google My Map, click "Share", ensure it's "Public", and extract the mid from the URL (e.g., mid=12345...).
